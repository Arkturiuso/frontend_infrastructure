1. После клонирования репозитория, следует перейти на ветку master.
2. Необходимо установить зависимости командой yarn install
3. После установки yarn автоматически будут активированы хуки
4. При пуше на ветку master запускается Github Actions, который собирает проек
5. После успешной сборки происходит автоматический деплой на Github Pages
6. После этого можно перейти на сайт по ссылке в на Github pages
